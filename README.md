# Discord-Face-Filter
ðŸ¤–: Face Filter is a face hiding effect using the cv2 and numpy modules. The effect supports a virtual camera for use in Discord

<h3>ðŸ“Œ: Instruction for use:

  1. Run the main.py script. (Your webcam will start working, and you'll see that the effect is already working perfectly)
  ![Face-Filter | Screenshot1 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen1.png)

  2. Now, let's link this effect to the Discord virtual camera. To do this, open OBS Studio and add "Window Capt"
  ![Face-Filter | Screenshot2 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen2.jpg)

  3. Next, in the Window panel, select the .exe python file([python.exe]: Face Filter) and click OK
  ![Face-Filter | Screenshot3 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen3.png)

  4. Great! Now you can see the effect in OBS. But the question remains. How to display it when you turn on the camera in Discord? To do this, click on Start Virtual Camera

  ![Face-Filter | Screenshot5 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen5.jpg)

  6. Next, go to Discord and select the OBS Virtual Camera in the camera settings
  ![Face-Filter | Screenshot6 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen6.png)

  7. Congratulations! Now you can use the effect for Discord hangouts
  ![Face-Filter | Screenshot7 ](https://github.com/sou1ton-exe/Discord-Face-Filter/blob/main/assets/screen7.png)
